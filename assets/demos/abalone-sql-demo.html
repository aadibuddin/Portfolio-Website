Average shell weight by collection region

Surfaces which coastal regions produced heavier-shelled abalone on average.

SELECT Water_region,
       ROUND(AVG(Shell_weight), 4) AS avg_shell_weight,
       COUNT(*) AS total
FROM abalone_data
GROUP BY Water_region
ORDER BY avg_shell_weight DESC
LIMIT 5;
